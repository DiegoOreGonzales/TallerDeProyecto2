3. PRUEBAS CON POSTMAN
================================================================================

Archivo de colección exportada:
  coleccion_postman.json

Descripción:
  Archivo JSON exportado desde Postman (Collection v2.1) que contiene
  todos los requests organizados en 7 carpetas lógicas con scripts de
  test automatizados para cada endpoint.

Documentación:
  docs/pruebas/GUIA_POSTMAN.md

Generación (pendiente — manual):
  1. Abrir Postman
  2. Importar la configuración de la colección descrita en GUIA_POSTMAN.md
  3. Configurar cada request con su método, URL, headers y body
  4. Pegar los scripts de test en la pestaña "Tests"
  5. Exportar como coleccion_postman.json
